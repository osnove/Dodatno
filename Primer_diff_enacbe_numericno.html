Uporaba numeričnih metod za reševanje diferencialnih enačb - prehodni pojav pri R L C vezjih¶


Dejan Kriaj, 2019

Namen: Zvezek (Notebook) je namenjen seznanjanju tudentov s prehodnim pojavom. Kako se pride do diferencialne enabe je opisano v ubeniku, tu pa je mogoe uporabiti numerino reevanje diferencialnih enab z uporabo Python knjinice $odeint$.

Knjinica $odeint$ uporablja ve metod za numerino reevanje diferencialnih enab.Najbolj preprosta je Eulerjeva metoda, kjer odvod $\frac{dx}{dt}=f(x)$ aproksimiramo kar z diferenco funkcije v doloenem asovnem koraku: $x(t+h)=x(t)+hf(x(t))$.

Bolj natanne so metode, ki odvod bolj natanno aproksimirajo. Na primer Euler-Richardsonova metoda, pri kateri se uporabi za odvod vmesne vrednosti $x(t+h)=x(t)+hf(x(t)+hf(x(t))/2)$. Ta metoda je znana tudi kot Runge-Kutta metoda drugega reda. e bolj pogosto se uporablja Runge-Kutta metodo etrtega reda ali tudi druge metode, odvisno od aplikacije. Funkcija $odeint$ si ob uporabi "sama" izbere najustrezneji nain reevanja.

Namen tega izvajanja ni predstavitev numerinih metod, pa pa njihova uporaba v konkretnih primerih analize prehodnih pojavov pri elektrinih vezjih. Za dodatne razlage numerinih postopkov, si lahko ogledate spodnje spletne strani, iz katerih sem izdelal primere.

Zaporedno R+C vezje¶

Vklop RC vezja na vir s harmoničnim signalom¶

Reujemo enabo ${{u}_{g}}=RC\frac{\text{d}{{u}_{C}}}{\text{d}t}+{{u}_{C}}$, kjer je ${{u}_{g}}(t)={{U}_{g}}\cos (\omega t+{{\varphi }_{g}})$.

Enabo zapiemo v obliki ${\frac{\text{d}{{u}_{C}}}{\text{d}t}=({u}_{g}}-{{u}_{C})/RC}$.

Zaetni pogoj bo odvisen od asa preklopa,saj velja $u_C(t=0^+)=u_C(t=0^-)$ .

## Vnos potrebnih knjinic
%matplotlib inline
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

## Vrednosti konstant
R=5e3
C=1e-7
RC=R*C
print(RC)
Ug=10
omega=2*np.pi*1000
fi=np.pi/2

def dUc_dt(u,t):   # Zapis diferencialne enabe v obliki funkcije, ki vrne izraun desne strani enabe, torej odvod napetosti po asu
    return (Ug*np.cos(omega*t+fi)-u)/RC

def plot_func(t,x):  # Priprava funkcije za izris
    fig = plt.figure(figsize=(12,5))
    axes = fig.add_subplot(1,2,1)
    axes.plot(t, x[:,0], 'r', label="$U_c(t)$")
    axes.plot(t, Ugt, 'b-', label="$U_g(t)$")
    plt.ylim([-1.2*Ug, +1.2*Ug])
    axes.set_xlabel("$t /s $")
    axes.set_ylabel("$U_C(t)$")
    plt.title('KONDENZATOR')
    plt.legend(loc='upper left')
    
    #izris napetosti na uporu - izraunan kot Ug(t)-uC(t)
    axes = fig.add_subplot(1,2,2)
    axes.plot(t,Ugt-x[:,0], 'r', label="$U_R(t)$")
    axes.plot(t, Ugt, 'b-', label="$U_g(t)$")
    axes.set_xlabel("$t /s $")
    axes.set_ylabel("$U_R(t)$")
    plt.title('UPOR')
    plt.show()

dt = 1e-5  # numerini asovni korak 
n_steps = 500  # tevilo korakov
t = np.arange(0, n_steps*dt, dt)  # asovni niz sestavljen iz n_steps s asovnim korakom dt
Ugt=Ug*np.cos(omega*t+fi)  # Ug(t) kot asovni niz
u = np.zeros((1)) # zaetni pogoj
u[0] = Ugt[0]    # Napetost na kondenzatorju ob preklopu
y = odeint(dUc_dt,(u[0]),t) # Zagon odeint funkcije, ki rei dif. enabo
plot_func(t,y)

0.0005
